\hypertarget{index_autotoc_md15}{}\doxysection{\texorpdfstring{Algorithm Overview}{Algorithm Overview}}\label{index_autotoc_md15}
A Scapegoat Tree is a self-\/balancing BST that maintains balance through periodic rebuilding\+:


\begin{DoxyItemize}
\item {\bfseries{α-\/weight-\/balanced}}\+: No subtree can exceed α × parent’s size (α = 2/3)
\item {\bfseries{Height bound}}\+: h ≤ log₁.\+₅(n), where n = number of nodes
\item {\bfseries{Lazy rebalancing}}\+: Rebuilds only when balance is violated
\end{DoxyItemize}

{\bfseries{Advantages}}\+:


\begin{DoxyItemize}
\item Simpler than AVL or Red-\/\+Black trees (no color/height metadata)
\item Amortized efficiency\+: rebuilds are rare, fast average-\/case operations
\item Space-\/efficient\+: minimal per-\/node overhead
\end{DoxyItemize}\hypertarget{index_autotoc_md16}{}\doxysubsection{\texorpdfstring{Complexity Analysis}{Complexity Analysis}}\label{index_autotoc_md16}

\begin{DoxyTable}{3}{}{}{1}
\SetCell{c,bg=\tableheadbgcolor,font=\bfseries}Operation  &\SetCell{c,bg=\tableheadbgcolor,font=\bfseries}Time Complexity  &\SetCell{c,bg=\tableheadbgcolor,font=\bfseries}Space  \\
Search  &O(log n) worst-\/case  &O(1)  \\
Insert  &O(log n) amortized  &O(1)  \\
Delete  &O(log n) amortized  &O(1)  \\
Rebuild  &O(n) occasional  &O(n) temporary  \\
Traversal  &O(n)  &O(n) for level-\/order  \\
\end{DoxyTable}
