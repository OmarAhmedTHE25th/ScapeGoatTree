A Scapegoat Tree is a self-\/balancing BST that maintains balance through periodic rebuilding:\+


\begin{DoxyItemize}
\item {\bfseries{Î±-\/weight-\/balanced}}:\+ No subtree can exceed Î± Ã— parent\textquotesingle{}s size (Î± = 2/\+3)
\item {\bfseries{Height bound}}:\+ h â‰¤ logâ‚.\+â‚…(n), where n = number of nodes
\item {\bfseries{Lazy rebalancing}}:\+ Rebuilds only when balance is violated
\end{DoxyItemize}

{\bfseries{Advantages}}:\+


\begin{DoxyItemize}
\item Simpler than AVL or Red-\/\+Black trees (no color/\+height metadata)
\item Amortized efficiency:\+ rebuilds are rare, fast average-\/case operations
\item Space-\/efficient:\+ minimal per-\/node overhead
