Scape\+Goat Tree Implementation

A self-\/balancing BST that maintains balance through periodic rebuilding.

Key Properties:\+
\begin{DoxyItemize}
\item α-\/weight-\/balanced:\+ No node\textquotesingle{}s subtree is heavier than α × parent\textquotesingle{}s subtree
\item α = 2/\+3 for this implementation
\item Height bound:\+ h ≤ log₁.\+₅(n) where n = number of nodes
\end{DoxyItemize}

Time Complexity:\+
\begin{DoxyItemize}
\item Insert:\+ O(log n) amortized, O(n) worst case during rebuild
\item Delete:\+ O(log n) amortized, O(n) worst case during rebuild
\item Search:\+ O(log n) worst case (tree stays balanced)
\end{DoxyItemize}
